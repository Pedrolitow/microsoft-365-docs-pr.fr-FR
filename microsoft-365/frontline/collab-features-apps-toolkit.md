# <a name="help-your-frontline-workers-use-collaboration-apps-and-features"></a>Aider vos employés de première ligne à utiliser les applications et fonctionnalités de collaboration

Maintenant que votre organisation a commencé à utiliser Microsoft 365 for frontline workers et Microsoft Teams, vous devez vous assurer que votre personnel est à l’aise avec les applications et fonctionnalités disponibles.

Utilisez votre [stratégie de communication d’entreprise](flw-corp-comms.md) pour partager ces ressources vidéo afin d’aider votre équipe de première ligne à tirer le meilleur parti des outils disponibles.

## <a name="tags-in-microsoft-teams"></a>Balises dans Microsoft Teams

L’étiquetage des utilisateurs et des équipes dans les publications et les conversations permet à vos employés de première ligne d’atteindre les bonnes personnes plus rapidement en recherchant des membres d’équipe par des attributs tels que le rôle, l’emplacement ou le projet.

[Formation des utilisateurs finaux pour les publications et les messages dans Teams](https://support.microsoft.com/office/create-and-format-a-post-e66777da-636b-49eb-9408-b0d88b212885) <br>
[Afficher la vidéo Balises dans Microsoft Teams sur YouTube](https://go.microsoft.com/fwlink/?linkid=2202727)
> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE54jBN]

## <a name="walkie-talkie"></a>Talkie-walkie

Walkie Talkie fournit une communication push-to-talk instantanée afin que vos employés de première ligne puissent communiquer facilement et en toute sécurité entre différents emplacements.

[Formation des utilisateurs finaux pour Walkie Talkie](https://support.microsoft.com/office/use-walkie-talkie-in-teams-884a008a-761e-4b62-99f8-15671d9a2f69) <br>
[Voir la vidéo Walkie Talkie sur YouTube](https://go.microsoft.com/fwlink/?linkid=2202710)
> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE546Fv]
