# <a name="remove-company-data-from-devices"></a>Supprimer des données d'entreprise sur des appareils

## <a name="remove-company-data"></a>Supprimer les données de l’entreprise

You can use Microsoft 365 for business to remove company data that your users have on their [devices](m365bp-app-protection-settings-for-android-and-ios.md) or [Windows PCs](m365bp-protection-settings-for-windows-10-devices.md) that are protected by Microsoft 365. **If you remove company data from a device, you cannot restore it later**. 
  
1. Accédez au Centre d’administration à l’adresse <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>.
    
2. Dans le menu de gauche, choisissez **Gestion**  \> **des appareils.**.
  
3. Sur la page **Gérer**, choisissez ou recherchez un utilisateur dont vous voulez supprimer les données, puis choisissez son nom. 
    
4. Dans le volet suivant, sélectionnez le ou les appareils dans la liste **Appareils**. Dans le volet de l'appareil qui s'ouvre, vous pouvez choisir de réinitialiser l'appareil aux paramètres d'usine ou de supprimer les données de l'entreprise, selon le type d'appareil. 
    
    ![Dans le volet de suppression des données de l'entreprise, sélectionnez l'appareil à partir duquel vous souhaitez supprimer les données.](./../media/resetorremove.png)
  
5. Dans le volet de confirmation, choisissez **Confirmer** \> **la fermeture**.
