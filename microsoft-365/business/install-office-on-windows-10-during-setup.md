# <a name="install-office-on-windows-10-during-setup"></a>Installer Office sur les appareils Windows 10 lors de l'installation

![Bannière pointant vers https://aka.ms/aboutM365preview .](../media/m365admincenterchanging.png)

Ce paramètre est éteint par défaut. Vous pouvez l’activer pour vous assurer que les utilisateurs ont la dernière version de Office sur tous leurs appareils Windows 10, mais vérifiez tout d’abord que tous les prérequis sont satisfaits. Les utilisateurs doivent être titulaires d’une licence Microsoft 365 Business Premium pour que ce paramètre place le logiciel sur l’appareil, en plus des autres conditions décrites dans La préparation du déploiement de [client Office](prepare-for-office-client-deployment.md)par Microsoft 365 Business Premium .
  
Nous vous recommandons d’accepter le paramètre par défaut lors de l’installation.
