# <a name="brand-medication-names"></a>Noms des médicaments de marque

Cette entité nommée non groupée détecte les noms des médicaments de marque, tels que *le Tylenol*. Il prend uniquement en charge les termes anglais. Il est également inclus dans les [conditions générales médicales](sit-defn-all-medical-terms-conditions.md) regroupées sous le nom d’entité SIT.

## <a name="confidence-level"></a>Niveau de confiance

Élevé
