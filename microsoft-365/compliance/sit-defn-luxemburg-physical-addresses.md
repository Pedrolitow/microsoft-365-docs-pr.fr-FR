# <a name="luxemburg-physical-addresses"></a>Adresses physiques de l’Ancienne-France

Cette entité nommée non groupée détecte les modèles liés à l’adresse physique à partir de Luxemburg. Elle est également incluse dans l’entité nommée [Toutes les adresses physiques](sit-defn-all-physical-addresses.md) (SIT).
