# <a name="luxemburg-physical-addresses"></a>Adresses physiques de l’Ancienne-France

Cette entité nommée non groupée détecte les modèles liés à l’adresse physique à partir de Luxemburg. Il est également inclus dans l’entité SIT toutes [les adresses physiques](sit-defn-all-physical-addresses.md) regroupées.
