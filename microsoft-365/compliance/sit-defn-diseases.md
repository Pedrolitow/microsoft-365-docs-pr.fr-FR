# <a name="diseases"></a>Maladies

Cette entité nommée non groupée détecte le texte qui correspond aux noms de maladies, tels que *le diabète*. Il prend uniquement en charge les termes anglais. Il est également inclus dans les [conditions générales médicales](sit-defn-all-medical-terms-conditions.md) regroupées sous le nom d’entité SIT.
