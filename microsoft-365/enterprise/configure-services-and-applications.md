# <a name="configure-microsoft-365-enterprise-services-and-applications"></a>Configurer Microsoft 365 Entreprise services et applications

Nos [instructions de mise en place](../admin/setup/setup.md) de base vous aident à faire en Microsoft 365 vos services et applications de base le plus rapidement possible. Il est parfois préférable d’obtenir des éléments configurés avant que tout le monde commence à les utiliser. Par exemple, si vous souhaitez configurer des stratégies de routage de messagerie, de stockage de fichiers ou de partage. 
  
Si vous souhaitez obtenir de l’aide Microsoft 365 configurer, utilisez **[](https://www.microsoft.com/fasttrack/microsoft-365)** FastTrack ou les guides d’installation [pour Microsoft 365 et Office 365 services.](setup-guides-for-microsoft-365.md)
