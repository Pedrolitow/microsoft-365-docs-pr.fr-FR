# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Obtenir la dernière version de l’analytique d’utilisation de Microsoft 365

L’application modèle peut être actualisée avec de nouvelles données ou de nouvelles visualisations plusieurs fois par an. Votre instance existante continuera de fonctionner, mais si vous souhaitez obtenir la dernière version, une nouvelle instance doit être créée et toutes les personnalisations doivent être appliquées à la nouvelle instance. Consultez [Activer l’analyse de l’utilisation de Microsoft 365](enable-usage-analytics.md).
